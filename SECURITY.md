## 报告安全漏洞

如果您发现安全漏洞，请**不要**通过公开的 GitHub Issue 报告。

请通过以下方式私下报告：

1. 发送邮件至项目维护者
2. 使用 GitHub Security Advisories
3. 在 PR 中描述漏洞详情

我们会尽快回复并处理。

## 安全最佳实践

WebOS 采用以下安全措施：

- **加密存储**: 使用 AES-256-GCM 加密敏感数据
- **密码保护**: 使用 PBKDF2 进行密码哈希
- **安全存储**: 敏感数据存储在 IndexedDB 中，使用加密保护
- **输入验证**: 所有用户输入都经过验证和清理
